1.安裝虛擬環境：sudo pip3 install virtualenv
2.進入虛擬環境：source ~/Documents/qtCart/dai2/bin/activate
3.安裝PyQt5:
pip3 install --upgrade pip
pip3 install --upgrade setuptools

sudo apt-get install qt5-default pyqt5-dev pyqt5-dev-tools
sudo apt-get install python3-pyqt5
sudo apt-get install python3-pyqt5.qtmultimedia

cp -rp /usr/lib/python3/dist-packages/sip* dai2/lib/python3.6/site-packages/
cp -rp /usr/lib/python3/dist-packages/PyQt5/ dai2/lib/python3.6/site-packages/

4.安裝需要的套件：pip install -r requirements.txt


測試是否正常啟動：
1.機器CPU設定：export OPENBLAS_CORETYPE=ARMV8
2.設定ip:ifconfig eth0 192.168.1.203 netmask 255.255.255.0 up
3.在 ~/Documents/qtCart資料夾下 執行 python Programe.py
